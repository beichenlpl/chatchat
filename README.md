# chatchat

#### 介绍
基于python开发的大模型调用基础框架


#### 使用说明

1.  修改配置文件
```commandline
cd config
vim __init__.py   # 在配置文件中添加大模型调用地址，模型名称，API_KEY等配置
```
2.  启动应用
应用启动分为两种模式，命令行模式和web模式
**命令行模式**
```commandline
python main.py cli
```
**web模式**
```commandline
python main.py api
```
